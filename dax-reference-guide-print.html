📅 Date Table (New Table → paste in DAX editor)
Mark as Date Table after creating: Table Tools → Mark as Date Table → select [Date]
Calculated TableRequired First
Date_Table =
ADDCOLUMNS(
    CALENDAR( DATE(2024, 1, 1), DATE(2027, 12, 31) ),
    "Year",           YEAR([Date]),
    "Month Num",      MONTH([Date]),
    "Month Name",     FORMAT([Date], "MMMM"),
    "Month Short",    FORMAT([Date], "MMM"),
    "Quarter",        "Q" & QUARTER([Date]),
    "Week Number",    WEEKNUM([Date]),
    "Day Name",       FORMAT([Date], "dddd"),
    "Is Weekend",     IF(WEEKDAY([Date], 2) >= 6, TRUE, FALSE),
    "Month Year",     FORMAT([Date], "MMM YYYY"),
    "Year Month Num", YEAR([Date]) * 100 + MONTH([Date])
)

Current PP # / Selected PP Label / PP Date Range
Display measures for card visuals and dynamic report headers.
Agent_Performance_SummaryMAX / SELECTEDVALUE / FORMAT
Current Pay Period # =
MAX( Agent_Performance_Summary[Pay Period#] )

Selected PP Label =
SELECTEDVALUE(
    Agent_Performance_Summary[PP Label],
    "Multiple Periods Selected"
)

PP Date Range =
VAR EndDate   = MAX( Agent_Performance_Summary[PPE Date] )
VAR StartDate = EndDate - 13
RETURN
    FORMAT(StartDate,"M/D/YYYY") & " – " & FORMAT(EndDate,"M/D/YYYY")
Answered — Current PP / Prior PP / Change
Period-over-period comparison. Swap [Answered] for any other SUM measure to replicate the pattern.
Agent_Performance_SummaryCALCULATE / VAR
Answered - Current PP =
VAR LatestPP = CALCULATE(
    MAX( Agent_Performance_Summary[Pay Period#] ),
    ALL( Agent_Performance_Summary ) )
RETURN CALCULATE(
    SUM( Agent_Performance_Summary[Answered] ),
    Agent_Performance_Summary[Pay Period#] = LatestPP )

Answered - Prior PP =
VAR LatestPP = CALCULATE(
    MAX( Agent_Performance_Summary[Pay Period#] ),
    ALL( Agent_Performance_Summary ) )
RETURN CALCULATE(
    SUM( Agent_Performance_Summary[Answered] ),
    Agent_Performance_Summary[Pay Period#] = LatestPP - 1 )

PP Answered Change =
[Answered - Current PP] - [Answered - Prior PP]
🔧 Formatting Helpers
Reusable patterns for converting seconds, dynamic titles, conditional colors, and filter context.
Format Seconds → H:MM:SS
For status durations (Available, Logged In, etc.). Swap VAR value for your measure.
Any TableVAR / FORMAT
Available Time Display =
VAR TotalSec = [Total Available (s)]   -- ← swap
VAR Hours    = INT( TotalSec / 3600 )
VAR Minutes  = INT( MOD( TotalSec, 3600 ) / 60 )
VAR Seconds  = MOD( TotalSec, 60 )
RETURN
    FORMAT(Hours,"0") & ":"
  & FORMAT(Minutes,"00") & ":"
  & FORMAT(Seconds,"00")
Format Seconds → M:SS
For call metrics like Avg Handle, Avg Talk (under an hour).
Any TableVAR / FORMAT
Avg Handle Time Display =
VAR TotalSec = [Avg Handle Time (s)]   -- ← swap
VAR Minutes  = INT( TotalSec / 60 )
VAR Seconds  = MOD( TotalSec, 60 )
RETURN
    FORMAT(Minutes,"0") & ":" & FORMAT(Seconds,"00")

Color Status Measure (Conditional Formatting)
Returns hex color based on threshold. Apply via Format → Conditional Formatting → Field value.
Any TableSWITCH / TRUE()
SL Color =
VAR SL = [Service Level %]
RETURN
SWITCH( TRUE(),
    SL >= 0.80, "#2ecc87",   -- green
    SL >= 0.65, "#f39c12",   -- amber
    "#C4272E"                 -- red (dept red)
)
